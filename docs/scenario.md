# Сценарий игры: Морской Бой ## Основной игровой процесс

Игрок участвует в пошаговом морском сражении против Искусственного Интеллекта. Поле боя представляет собой две сетки 10x10. Игрок скрытно расставляет свой флот, после чего начинается фаза активных боевых действий.

**Цель игры:** Полностью уничтожить флот противника, сохранив как можно больше собственных кораблей и заработав максимальное количество очков.

## Фазы игры

1.  **Placement (Расстановка):** Игрок размещает корабли разной длины (от 1 до 4 палуб). Система валидации контролирует соблюдение дистанции в одну клетку между судами.
2.  **Battle (Битва):** Игроки по очереди наносят удары. При попадании в цель игрок получает право на дополнительный ход.
3.  **Result (Итог):** Фиксация победы, подсчет бонусных очков и переход на новый уровень сложности.
