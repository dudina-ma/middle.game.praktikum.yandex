# Спрайты кораблей

Сюда нужно добавить изображения кораблей, которые использует игра.

Ожидаемые файлы (пути заданы в `GAME_CONFIG.shipSprites`):

- `ship-1.png`
- `ship-2.png`
- `ship-3.png`
- `ship-4.png`
- `hit.png` — крестик для отображения попадания

Рекомендуемые размеры спрайтов (ориентация «ряд» — горизонтально):

- Ширина: `CELL_SIZE.x * длина + DIVIDER_W * (длина - 1)`
- Высота: `CELL_SIZE.y`

Для вертикальных кораблей спрайт автоматически поворачивается на 90° при отрисовке — отдельные файлы не требуются.

Спрайт попадания (`hit.png`): рекомендуется размер `CELL_SIZE.x × CELL_SIZE.y`.

Все изображения должны иметь прозрачный фон, чтобы корректно накладываться на поле.
